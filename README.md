# 5dplinko.app

Five-dimensional multiplayer **pot Plinko** on [ProofNetwork](https://proofnetwork.lol).

|--|--|

Bet SOL → 5D VRF path → 9 bins → payout from shared pot (capped to cash; max mult ×5).

**Mults:** `[5, 3, 2, 1.2, 0.6, 1.2, 2, 3, 5]`
